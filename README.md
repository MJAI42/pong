# Classic Pong Game 🏓

A minimalist recreation of the 1972 Atari classic. This Pong clone features smooth 2D physics, a crisp retro aesthetic, and local multiplayer. Distilling gaming to its purest form—two paddles and a ball—it’s perfect for a nostalgia hit or exploring the fundamentals of game loop logic. Simple to learn, impossible to put down.

## Features

- **Local Multiplayer:** Play against a friend on the same keyboard.
- **Classic Gameplay:** Retro style aesthetics with classic sound effects.
- **2D Physics:** Smooth and responsive ball and paddle movements.
- **Keyboard Controls:** Simple and intuitive controls.

## Prerequisites

- [Python 3.8+](https://www.python.org/downloads/)

## How to Play

Run the main game file to start the game:

```bash
# Example command, adjust according to the main python script
python main.py
```

### Controls

- **Player 1 (Left Paddle):** `W` to move up, `S` to move down.
- **Player 2 (Right Paddle):** `Up Arrow` to move up, `Down Arrow` to move down.
- **General:** `Space` to start the game / pause, `ESC` to quit. *(Customize these according to your code)*
